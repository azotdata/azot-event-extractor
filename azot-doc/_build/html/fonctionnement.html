Fonctionnement d’AZOT

Comment l’installer?

Les pré-requis indispensables sont les suivants:

  • newspaper (pour le web scraping)
  • nltk (pour le traitement des textes)
  • système de gestion de base de données No-SQL (mongodb ou couchdb)
Cela étant fait, le projet peut être cloné ici [lien du git].

Comment le lancer?

Azot est à la fois pour la collecte et l’analyse de données textuelles. Ces scripts sont de ce fait bien distincts. La collecte est lancée autant de fois que possible en background, afin d’avoir les informations en temps réel émises par les sites de presse en ligne. La classification est lancée deux à trois fois par semaine pour garder la cohérence des évènements.
